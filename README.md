# 🏗️ Forgewell Industries Landing Page

A modern and responsive industrial company website built using **HTML, CSS, and JavaScript**, featuring service showcases, project highlights, company statistics, testimonials, and a contact form.

* Responsive Design
* Modern UI/UX
* Mobile Navigation
* Service & Project Sections
* Contact Form
* Accessibility Support (ARIA)
* SEO-Friendly Structure

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
